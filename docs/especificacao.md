# SimpleDL Requirements

## System

- Windows 10/11
- Python 3.10+

## Python packages

Install all dependencies:

```bash
pip install -r requirements.txt
```

## External dependencies

The bootstrap installs and configures automatically:

- yt-dlp
- FFmpeg (compatible with yt-dlp)

No manual FFmpeg installation is required when using the bootstrap.

## Optional

- Deno (recommended for YouTube JS challenge solving)
